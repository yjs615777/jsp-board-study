# jsp-board-study
JSP &amp; Servlet 기반의 레거시 게시판 연습 프로젝트

프로젝트 개요
본 프로젝트는 공공기관에서 실제로 운영 중인 레거시 웹 환경을 이해하고,
현장 소스코드 이해 역량을 강화하기 위해
JSP/Servlet 기반 웹 애플리케이션으로 개발하였다.

프로젝트 정보

프로젝트명: MAISON (메종)
개발 기간: 2026.01~
프로젝트 유형: 패션 브랜드 검색 및 정보 제공 웹 애플리케이션
개발 인원: 1인 (Full Stack)

프로젝트 목적
SPA 패션 브랜드에 대한 정보를 체계적으로 제공하고, 사용자가 원하는 브랜드를 쉽게 검색하고 탐색할 수 있는 플랫폼 구축
